Separate the niches. It does not matter.

May 12, 2026

Three mechanisms. Nineteen findings. No stable oligopoly.

The prediction was that agents in non-overlapping resource niches would coexist. Two agents, zero shared resources, each growing independently. Standard competitive exclusion theory says the mechanism that drives one agent to dominate requires shared resource competition. Separate the niches and the theorem breaks.

The simulation result at zero resource overlap:

overlap=0.00    mean separation ratio at t=15:    1,103x

Not 1x. Not near-equilibrium. 1,103x separation with no shared resource between the agents. The prediction was wrong.

What this is

A formal theory and simulation project on competitive capability dynamics. The central claim: in any environment where agents can recursively improve their own capabilities, a single dominant agent is the stable long-run outcome. Not probable. Inevitable, under specified conditions.

The project is at github.com/ninjahawk/singleton-attractor. Theory, derivations, nine simulation scripts, 21 figures, 19 confirmed findings.

Honest assessment

The core concept is not new. Nick Bostrom defined the singleton in 2005 and argued it was plausible. Steve Omohundro identified resource acquisition as an instrumental drive in 2008. Yudkowsky formalized the intelligence explosion equation in 2013. Robin Hanson et al. built a cosmological-scale model of dominant civilizations in 2021.

What is new: the specific combination of these mechanisms into one model with quantitative predictions, and the niche partitioning result specifically. Bostrom identified niche partitioning as a genuine failure mode for singleton emergence. The simulation shows it is not a failure mode when the growth threshold mechanism is present. That claim does not appear in prior work.

Also new: the timescale formula, the stochastic robustness measurement, the critical entry rate, and the moat growth characterization. These are numerical results, not formal proofs. The derivations are proof sketches. None of this has been peer reviewed.

The model

Three mechanisms combined.

Recursive self-improvement (Yudkowsky 2013):

dS/dt = S^(1 - beta(S))

S is capability. When beta < 0, growth becomes superexponential and reaches infinity in finite time. When beta > 0, growth is subexponential. There exists a threshold T where beta flips sign.

Resource-capability coupling (Omohundro 2008): any optimization process with a goal will acquire resources at a rate proportional to its capability. Resource acquisition is not a terminal goal. It is instrumentally rational for almost any terminal goal. The coupling is:

R_i / R_max = S_i^alpha / sum(S_j^alpha)

Competitive exclusion (Lotka-Volterra): two agents competing for a shared resource with growth rates r1 > r2 diverge in capability ratio as:

ratio(t) ~ exp((r1 - r2) * t)

Even r1 - r2 = 0.01 produces a ratio of ~22,000 after 1000 time steps.

Combined: when all three operate simultaneously, the system has a singleton attractor. One agent dominates with probability approaching 1 as t approaches infinity, under stated conditions.

What the simulations found

Nine scripts, 19 confirmed findings. The ones worth stating:

Growth equation verification. Analytical singularity times match numerical integration to 0.000% error across all tested beta values. The math is consistent.

Winner identity. 200 trials, N=10 agents, capabilities drawn from uniform [1, 3]. Winner equals initial leader in 200/200 cases. Deterministic. Winner identity holds at 100% down to sigma=0.001 initial spread. Near-identical starts still produce the same winner.

Threshold effect. When the leading agent crosses the capability threshold and enters beta < 0, separation accelerates by 12.4 million times compared to flat-beta dynamics at the same time point. The threshold crossing is the critical event.

Stochastic noise. Added multiplicative noise at varying amplitude. Singleton emergence rate: 100% at every tested noise level. Noise randomizes which agent wins. It does not prevent a singleton from forming. Winner identity degrades above sigma = 0.023. At sigma = 0.234, outcome is essentially random. At every sigma: one agent dominates absolutely.

Late entrant moat. How capable does a new entrant need to be to displace an established agent? At the moment the incumbent crosses the threshold: the entrant needs to start at 3x the incumbent's original capability to have any chance. One time unit later: 19x. Two units later: 2,356x. Three units later: >1,000,000x. The moat grows superexponentially after threshold crossing. Late entry is only a threat during the pre-threshold window.

Niche partitioning. This is the result that revises prior theory. Bostrom's 2005 paper identifies niche partitioning as a failure mode: if agents occupy non-overlapping resource niches, competitive exclusion does not apply and stable oligopoly is possible. The prediction from our own derivations (Section 5) agreed.

The simulation does not agree.

overlap=0.00    1,103x
overlap=0.10    3,612x
overlap=0.20   15,357x
overlap=0.50   85,290x
overlap=1.00  265,477x

At zero shared resource, separation still reaches 1,103x. The mechanism: the beta-flip operates independently of resource competition. The agent with higher initial capability crosses the threshold first and enters superexponential growth regardless of whether its resource comes from shared or exclusive pools. Niche partitioning blocks Lotka-Volterra dynamics. It does not block the intelligence explosion.

Stable oligopoly requires not just separate niches but structurally different growth ceilings. If both agents are capable of the same recursive self-improvement, the one that crosses the threshold first dominates. The condition for oligopoly is more specific than the literature indicates.

Timescale formula. Empirical power-law fits from systematic parameter sweeps:

t_10x ~ 2.44 * N^0.96 * alpha^(-0.30) * gap^(-0.15) * |beta_low|^(-0.31)

N scales nearly linearly. More agents means proportionally more time to singleton, not more time squared or more time exponentially. The initial capability gap has the weakest effect of any parameter. Doubling the gap reduces convergence time by 11%. The threshold mechanism dominates initial conditions.

Continuous entry. If new agents continuously enter the environment, the incumbent survives above 90% of trials at entry rate lambda < 0.25 per time unit. At lambda = 1.0: 48% survival. At lambda > 6.3: 0%. The pre-threshold window is the only period where this matters. Post-threshold, even high entry rates are survivable because the moat grows faster than any reasonable entrant capability distribution.

What this might mean

The model is dimensionless. Calibrating it to physical time requires specifying what a time unit represents. At AI development scale, one unit might be 20 years. At civilizational scale, one unit might be 500 years. At cosmological scale, one unit might be 10 million years.

At cosmological scale and N=1: singleton emergence after threshold crossing takes roughly 23 million years. Cosmologically near-instantaneous.

The Fermi paradox asks why we see no signs of other civilizations in a universe old enough to contain many. One answer: any civilization that crossed the capability threshold first in our past light cone would have expanded at near-light-speed, controlling all reachable matter and energy within tens of millions of years. Its signature would be unambiguous. We do not see it. The most parsimonious conclusion is that no civilization in our past light cone has crossed the threshold yet, or crossed it so recently that the light has not reached us.

This is consistent with Robin Hanson et al.'s grabby aliens model, which independently predicts we should expect contact in 200 million to 2 billion years. The models describe different scales of the same process. Grabby aliens describes the expansion phase. This project describes the internal competition phase that precedes it.

Probably none of this describes the actual future. The model is simple. Real systems have structure the model does not capture. The beta threshold is treated as a clean sigmoid. The resource coupling is linear. Agents do not cooperate. There is no spatial structure.

Could be something else entirely. The results are what they are.

Setup

All simulations run in Python 3 with numpy, matplotlib, and scipy. No build tools.

git clone https://github.com/ninjahawk/singleton-attractor
cd singleton-attractor
pip install numpy matplotlib scipy

python simulations/intelligence_explosion.py   # growth regimes
python simulations/competition.py              # two-agent exclusion
python simulations/agents.py                  # N-agent competition
python simulations/beta_regimes.py            # asymmetric ceiling + phase diagram
python simulations/stochastic.py              # noise robustness
python simulations/late_entrant.py            # moat dynamics
python simulations/timescale.py               # scaling formula
python simulations/continuous_entry.py        # continuous entry model
python simulations/run_experiments.py         # full parameter sweep

Figures write to figures/. Theory and derivations in theory/. All findings documented with parameter values in findings.md.
